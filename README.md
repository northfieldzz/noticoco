# noticoco

Virtual Youtuber 桐生ココの動向を知らせるLine bot

## development

```shell script
$ go version
go version go1.14.2

$ go mod download
$ go run server.go
```

## deployment

### Environment

- API_KEY  
  GCP youtube data apiの認証APIキー

- API_VERSION  
  このAPIのバージョン指定

- PORT  
  リスニングポート指定

- COCO_CHANNEL_ID  
  桐生ココ チャンネルID

- LINE_REPLY_URL  
  Line Webhook URL

- CHANNEL_SECRET  
  Line Webhook Secret Key

- CHANNEL_TOKEN  
  Line Webhook Token

- ROOM_ID  
  メッセージをプッシュするline id
